# 1. 市场现状

## 为什么现在是最好的时机

**全球信号：** OpenClaw已经有214K GitHub Stars。中国小红书上代装市场火爆 — ¥300-800一次安装，¥9.9-39.99的课程卖几千份。腾讯云都在搞免费装机活动，一次给800人部署。

**新西兰空白：** 新西兰没有人做这个。整个澳新市场是空白的。零竞争。

**NZ华人市场：** 2026年预计336,600人，69%集中在奥克兰。高度连接中文社交媒体，刷小红书看到OpenClaw火爆，但本地找不到中文+本地时区的服务商。

**关键判断：安装不是机会，安装之后才是。**

中国的代装市场已经在被云厂商打价格战。真正的机会是装完之后的场景配置、持续调优、教人用起来、定制开发。

- 每天在生产环境跑OpenClaw — 不是纸上谈兵
- 中英双语 — 覆盖整个NZ市场

**画像：** 25-45岁，中文为主，奥克兰，在小红书看到OpenClaw但自己装不上或装完不会用。

**高价值痛点（每个都能量化省多少钱）：**

| 场景 | 痛点 | AI价值 | 替代方案成本 |
|------|------|--------|-------------|
| 英文合同解读 | 租房/雇佣/保险合同看不懂，签了踩坑 | 逐条翻译+标风险点 | 律师$300-500/小时 |
| 保险/贷款比较 | 车险差$500/年，贷款利率差0.5%=$数万 | 条款对比+最优方案 | 理财顾问$150-300/小时 |
| 签证管理 | 不只提醒到期，帮准备材料+跟踪+解读政策 | 全流程辅助 | 移民顾问$200-500/小时 |
| 税务优化 | 哪些能抵扣、WFH claim、GST — 搞错多交几千 | 提前整理+优化建议 | 会计$150-250/小时 |
| ACC claim | 很多人不知道能claim什么，白丢$1K-$10K+ | 判断+准备+填表+跟踪 | ACC advisor收费 |
| 房产文档 | LIM报告、建筑报告、买卖合同 — 读不懂=盲买 | 解读关键内容+标风险 | 律师审合同$2,000+ |
| 副业/freelance管理 | 发票、记账、税务、客户跟进一人做 | 全套管理自动化 | 记账$50-80/小时 |
| 消费大决策 | 换电力/宽带/保险 — 每个决策省$300-3,000/年 | 比较分析+推荐 | 自己研究5-10小时/项 |

- ❌ 简单翻译 — Google Translate够了
- ❌ 健身打卡/菜谱管理 — 太trivial

**获客渠道：** 小红书 → 微信 → WhatsApp
**入口套餐：** 个人版（从$99/月起）或 生活版

# 10. 付费推广策略（薯条）

## 为什么用薯条不用聚光
- **聚光**需要企业号，起投高，适合成熟品牌
- **薯条**个人号即可使用，¥50起投，3.0版已取消500粉丝门槛，新号友好

## 预算：¥100/周（~¥430/月）

### 第1-2周 — 测试期（¥100/周 拆成2笔）

| 步骤 | 预算 | 操作 |
|------|------|------|
| 第一笔 | ¥50 | 投当周自然互动最好的1篇帖子，模式选「阅读量」，投放6小时 |
| 观察 | — | 1小时后看点击率：>7%继续，<3%立刻改封面标题，不盲目续投 |
| 第二笔 | ¥50 | 数据好→追加同一篇；数据差→换另一篇表现好的 |

**模式选择：** 新号优先「阅读量」— 先要曝光基数

### 第3-4周 — 优化期（¥100/周）

| 调整 | 说明 |
|------|------|
| 只投点击率>7%的帖子 | 不浪费钱在差帖上 |
| 模式切到「粉丝关注量」 | 开始积累精准粉 |
| 每周最多投2篇 | 集中火力，不撒胡椒面 |

### 投哪类帖子 ROI 最高

| 优先级 | 内容类型 | 理由 |
|--------|---------|------|
| 🥇 | ⚠️避坑类 | 天然高点击率，"NZ华人常踩的X个坑" |
| 🥈 | 💰省钱/防亏类 | 转化最直接，"合同没看AI帮你看，省$2,000" |
| 🥉 | 📸案例展示 | 建立信任，"帮XX老板装完的效果" |
| ❌ | 🌿生活类 | 互动低转化差，不投 |

### 投放操作要点

1. **发帖后等2天再投** — 先看自然数据判断帖子质量
2. **投放后1小时内补3条评论** — 引导互动，提升算法推荐
3. **投放时间选白天** — NZ下午=中国上午，华人活跃时段
4. **人群定向：自定义 > 智能** — 选性别、年龄、兴趣标签
5. **数据判断标准：**
   - 点击率>7% → 好帖，追加
   - 点击率3-7% → 一般，优化封面后再测
   - 点击率<3% → 停投，换帖

- ¥100薯条 ≈ 5,000次曝光
- 好内容+薯条 → 撬动10倍自然流量
- 薯条单独：+35%阅读（一般）
- 薯条+内容优化组合：+200%阅读（最佳）
