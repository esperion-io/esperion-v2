# Esperion Business Plan — V5 Changes Summary

**Version:** V4.1 → V5.0  
**Updated:** March 2026  
**Key theme:** Global domain strategy. No "NZ" in domain names. Standalone microsites, not redirects.

---

## What Changed in V5

### 1. Section 8 — SEO & Domain Strategy (Complete Rewrite)

**The problem Anthony identified:** Do people actually search "AI implementation NZ" or just "AI implementation"?

**Answer: Mostly just "AI implementation."**

Why this matters:
- Google geo-targets by IP. Someone in Auckland searching "AI for small business" gets NZ results automatically. No "NZ" needed in the query.
- The `.co.nz` TLD already signals NZ to Google's local algorithm — adding "NZ" to the domain name is redundant.
- "NZ" in domain names blocks AU expansion. `aifortrades.com` works in Sydney. `aifortrades.co.nz` doesn't.
- Broader terms (without country modifier) have 10–50× more search volume than country-modified searches.

### 2. New Domain Names (All "NZ" Removed)

**Old V4 domains → New V5 domains:**

| Old (V4) | New (V5) | Why |
|----------|----------|-----|
| `aifornz.com` | *(dropped)* | "NZ" in name, redundant |
| `aifortrades.co.nz` | `aifortrades.com` | Global .com, works for AU |

**Fallbacks if `aiforbusiness.com` is taken:** `aiimplementation.com` or `getaibuilt.com`

**Total domain cost: ~$60–80/year NZD** (down from $85–100 in V4)

### 3. Standalone Microsites — Not Redirects (Critical Architecture Change)

**V4 approach:** Buy domains, redirect to esperion.io  
**V5 approach:** Each domain is a full standalone microsite

Pure 301 redirects provide almost zero SEO value. A parked domain that redirects doesn't rank — Google sees through it.

**Each microsite has its own:**
- Content targeting its own keyword cluster
- ROI tools, case studies, demo booking
- Independent ranking potential
- Link back to esperion.io (for brand trust, not redirect)

**Microsite architecture:**
```
aifortrades.com      → "AI for Trade Businesses" — trades ROI, testimonials, demo booking
aiforbusiness.com    → "AI Implementation for Businesses" — blog, ROI calc, case studies
openclawsetup.com    → "OpenClaw Setup & Implementation" — educational + CTA
esperion.io          → Brand home — company story, team, all verticals, trust
```

### 4. Phased Microsite Build (New)

- **Week 1:** Buy all domains (~$60–80/yr). Basic "coming soon" with email capture.
- **Month 1:** Build ONE microsite — the winning vertical's domain only.
- **Month 2:** Add second microsite if dual vertical is working.
- **Month 3+:** Build remaining as content + case studies accumulate.

Don't build all microsites at once. Build what matches the traction signal.
