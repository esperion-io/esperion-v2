# 📋 PLAN RECOMMENDATION GUIDE (Internal — For Anthony)

## How to Map Questionnaire Answers → Plan Tier

### Personal Use → Plan Mapping

| Profile | Recommended Plan | Price Point |
|---------|-----------------|-------------|
| Basic needs (calendar, reminders, simple tasks), beginner tech level, budget <$150/mo | **Personal Starter** | From $99/month |
| Multiple needs, intermediate tech, wants messaging integration, budget $150-300/mo | **Personal Pro** | From $199/month |
| Power user, advanced tech, multi-channel, voice calling, smart home, budget $300+/mo | **Personal Premium** | From $299/month + setup |

### Business Use → Plan Mapping

| Profile | Recommended Tier | Price Point |
|---------|-----------------|-------------|
| Sole trader / 1-5 employees, 1-2 use cases, basic integration, budget-conscious | **Tier 1** | $2,000–$3,000 setup + $500–$750/month |
| 6-19 employees, multiple use cases, CRM/software integration needed, moderate budget | **Tier 2** | $4,000–$6,500 setup + $1,000–$1,625/month |
| 20+ employees OR complex multi-system integration OR enterprise-grade requirements | **Tier 3** | $8,000–$14,000 setup + $2,000–$3,500/month |

### Key Qualifier Questions (Decision Drivers)

- **Q8 (employees)** → Directly maps to tier level
- **Q10 (pain points)** → More pain points selected = higher tier (more systems to build)
- **Q11 (enquiry volume)** → 100+/week = needs robust system = Tier 2-3
- **Q14 (AI wish list)** → More items = higher scope = higher tier
- **Q20 (existing software)** → More integrations needed = higher tier
